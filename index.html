Moodle - KAMAR

The Moodle - KAMAR integration provides live integration between your schools KAMAR data, and the open-source Moodle Learning Management System (LMS). Its implementation is closely based on the KAMAR Parent Portal but for student use only (there is no parent access provided by Moodle-KAMAR).

Information provided by the Moodle - KAMAR block includes:

  • Daily Notices
  • Timetable
  • Details
  • Timetable
  • Subject Results
  • NCEA Summary
  • All Results
  • Groups
  • Reports
  • Awards
  • Attendance
  • Pastoral
  • Fees
  • Careers
  • Interviews
  • Course Selection All that is required is that your students Moodle username match the username stored in KAMAR

Active Directory Management

Active Directory Management Service

The Active Directory Management Service (ADMS) is a tool to enable your SMS to populate Microsoft Active Directory with users and groups that exist within your school’s instance of KAMAR. It was previously know as KAMAR Directory Manager Service (KDMS) and is a replacement for the old “Network Users”
